# Render some osu! stats as an image!
Image doesn't update dynamically, so please make a new card whenever your stats change
#.env setup
your env should look like this:
```
OSU_CLIENT_SECRET="your_osu_client_secret_(https://osu.ppy.sh)"
IMGBB_KEY="your_imgbb_key(https://api.imgbb.com)"
IMG_USER_ID="your_hcti_uid(https://htmlcsstoimage.com/dashboard/api-keys)"
IMG_API_KEY="your_htci_key(https://htmlcsstoimage.com/dashboard/api-keys)"
```
